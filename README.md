# chatigo-social-network


## Для запуска проекта необходимо:

```
Frontend:
1. Установить Node.js;
2. В папке выполнить установку Sock.js и Stomp.js при помощи команд: `npm install react-stomp` и `npm install sockjs-client`;
3. Запустить при помощи `npm start`;

Backend:
1. На примере IDE Eclipse установить версию для работы с Java EE;
2. В классе `DataBaseConnection.java` изменить переменную `photoPath` на собственный путь к папке `img` в корне проекта;
3. Запуск осуществляется в интерфейсе IDE;
```
